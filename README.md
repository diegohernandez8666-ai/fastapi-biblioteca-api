# FastAPI Proyecto

API desarrollada con FastAPI que permite gestionar usuarios y subir imagenes.

## Instalacion

1. Clonar repositorio

git clone URL_DEL_REPO

2. Instalar dependencias

pip install -r requirements.txt

3. Ejecutar servidor

uvicorn app.main:app --reload

## Documentacion

http://127.0.0.1:8000/docs

## Funcionalidades

- CRUD de entidades
- Base de datos SQLite
- Cache con Redis
- Subida de imagenes
